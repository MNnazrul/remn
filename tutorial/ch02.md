In Rust, is_control() is a method for char values that checks whether a character is a control character.

Control characters are non-printable characters that do not represent a written symbol but rather control how text is processed. Examples include:

'\n' (newline)

'\r' (carriage return)

'\t' (tab)

'\0' (null character)
